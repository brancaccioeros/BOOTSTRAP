BTS_1.1 - Bootstrap Get Started

1. A cosa serve Bootstrap?

Bootstrap è un framework open-source per lo sviluppo di interfacce web responsive e mobile-first. Fornisce componenti HTML, CSS e JavaScript già pronti per creare layout moderni in modo semplice e veloce.

2. Cosa devi importare per creare una pagina web con Bootstrap?

Esempio di codice da inserire nella <head> della pagina:

<meta name="viewport" content="width=device-width, initial-scale=1">
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js"></script>

3. Cos’è un CDN? Che vantaggi comporta?

Un CDN (Content Delivery Network) è una rete distribuita di server che consente di caricare risorse (come Bootstrap) più velocemente. I vantaggi includono:

  • Caricamento più rapido grazie ai server distribuiti
  • Non è necessario scaricare Bootstrap e inserirlo nel progetto
  • Meno spazio occupato nei file locali

4. A cosa serve questo attributo?

content="width=device-width, initial-scale=1"

Serve a rendere la pagina responsive: imposta la larghezza della pagina alla larghezza del dispositivo e il livello di zoom iniziale a 1.

5. Differenza tra Container e Container-fluid?

Container: ha una larghezza massima fissa e si adatta automaticamente al breakpoint corrente.
Container-fluid: occupa sempre il 100% della larghezza disponibile, su qualsiasi dispositivo.
